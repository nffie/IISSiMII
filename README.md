## Практическая работа 1: Сравнение методов машинного обучения

### Задания

#### Линейный фильтр
- Разобран код линейного классификатора.
- Протестирован на данных из таблицы 1.
- Придуманы 4 тестовые точки для проверки способности модели предсказать метки классов.

#### Логистическая регрессия
- Разобран код логистической регрессии.
- Протестирован на данных из таблицы 1.
- Придуманы 4 тестовые точки для проверки способности модели предсказать метки классов.

#### Нейронные сети с использованием scikit-learn
- Загрузка датасета о вине из библиотеки scikit-learn.
- Построение модели MLPClassifier с использованием стохастического градиентного спуска.

### Ответы на вопросы
1. **Градиентный спуск** — _метод оптимизации, при котором обновления параметров модели происходят в направлении наибольшего убывания функции потерь._
2. **Линейный классификатор** — _модель, которая использует линейную комбинацию признаков для разделения данных на два класса._
3. **Логистическая регрессия** — _модель для классификации, использующая сигмоидную функцию для предсказания вероятности._
4. **Нейронная сеть** — _сложная модель для обработки данных, состоящая из нескольких слоев, включая скрытые слои._
5. **Персептрон** — _простой тип нейронной сети, состоящий из одного слоя нейронов._
6. **Метод обучения многослойного персептрона** — _обратное распространение ошибки (backpropagation)._
